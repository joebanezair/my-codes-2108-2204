# laiwan_io_web
提供来玩 web 资源支持

## 功能介绍
+ 来玩首页
+ 用户协议: {{domain}}/aggrement.html
+ 隐私政策: {{domain}}/privacy.html

## 使用 React 开发

搭建 React 开发环境, 安装 node, yarn

### 启动开发环境
```shell
$ cd react_laiwan_com
$ yarn
```

## 跑 Web
```shell
$ cd react_laiwan_com
$ yarn start
```
## 如何发布到 production
```shell
$ git checkout master
$ ./make_tag
```
